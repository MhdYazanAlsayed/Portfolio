2. Available Badge Colors

badge-lavender badge-cyan badge-gold badge-magenta badge-lime badge-holographic badge-neural badge-quantum

3. Date Format

Use YYYY-MM format for dates (e.g., "2024-01", "2024-12")

4. Skills Structure

Each skill can have:

  • name - The skill name
  • type - Category (language, framework, tool, etc.)
  • badge - Badge color class

üîß JavaScript API

The system provides a JavaScript API for advanced usage:

// Access the experiences manager
const manager = window.experiencesManager;

// Add a new experience
manager.addExperience({
  id: 4,
  title: "New Job",
  company: "New Company",
  // ... other properties
});

// Update an existing experience
manager.updateExperience(1, {
  title: "Updated Title"
});

// Remove an experience
manager.removeExperience(1);

// Reload experiences from JSON
manager.loadExperiences().then(() => {
  manager.renderExperiences();
});
